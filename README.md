# Moda
WIP Open Source Samsung Monte (and other shp devices) user interface made in flash player

# How to use?
on a cfw flashed Samsung Monte or similiar, put the RES folder located in /FlashSource into the Root of the Flash.

to compile, use Adobe Flash CS4 Proffesional (cs5 should be fine too)
